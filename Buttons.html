Buttons

The system design includes one primary button, with three states.

  • Primary Button: normal state
  • Primary Button: hover state
  • Primary Button: inactive state

Accessibility

Buttons must be created with size and color contrast that is visible to all users. The text must be dcesrciptive of what the button is for. The button must react so as to alert the user of its use (i.e.hover state, inactive state).

Usage

  • Used to initiate actions
  • Used for page navigation in header
  • Must contain labels - descriptive text

Code

Primary Button: normal state

<button type="button" class="btn btn-primary"> Descriptor </button>

Primary Button: hover state

<button type="button" class="btn btn-primary"> Descriptor </button>

Primary Button: inactive state

<button type="button" class="btn btn-inverse"> Descriptor </button>
