Navigation Bar & Footer

The navigation bar and footer should be included on all pages, and adjust per device and screen size.

Navigation Bar

Accessibility

The nvigation bar should include accessibile colors chosen from the color palette of the design system. All colors and fonts included should take into consideration visual contrast for every user. Within the navigation bar, each button should be descriptive to its action.

Usage

  • Must be used in each web page.
  • Must adjust to every device and screen size.
  • Must contain accessible colors and fonts.

Code

Below is an example of the code that can be used to create this navigation bar.


                <nav class="navbar">
                  <div class="container-fluid">
                  <a href="typehierarchy.html">
                  <img src="../img/Logo3.png" class="card-img-top image-item2" alt="Logo Icon">
                  </a>
                  <button type="button" class="btn btn-primary1 align-left"> WORK </button>
                  <button type="button" class="btn btn-primary1"> ABOUT </button>                    
                  </div >
                  </nav >

Usage
